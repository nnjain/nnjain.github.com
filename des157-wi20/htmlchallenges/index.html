Why we use the Normalize CSS Script

Normalise CSS Script is a small CSS file that ensures better cross-browser consistency. We use it because it maintains the broswer defaults without erasing them and corrects any inconsistencies that might be present between different browsers. Moreover, it also improves usability by correcting subtle improvements.

What is Semantic Markup

Semantic markup is using HTML tags that tells the browsers about the content. It is important for creating content that is both human and computer readable.The semantic elements are important because they help clean up the document structure by offering more information than non-semantic tags.
