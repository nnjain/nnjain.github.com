About Media Queries

Media query uses the media rule to include a block of CSS styling rules to only if a certain condition is true. It is important for responsive design which means creating web pages that dynamically change in response to different screen sizes. The min-width and the max-width are just as they sound. When we define min-width it means that the designing rules will be applied when the screen size is equal to or more than the width defined. For max-width, on screens that are the defined width or less, apply the certain rules. We should design mobile first because due to the less screen space, we will focus on the vital aspects of the website. This will then impact our design for tablets and desktops. The order goes smaller-then-larger.

Relative Units

A relative unit in CSS means a specific elememt gets sizing in relation to some other element. In font relative length, units can be defined as em, ex, ch and rem. There is also a % value, which is always relative to another value. Using relative values means that things can scale up and down according to some other value.
