Inheritance in CSS

Inheritance in CSS means that some CSS properties inherit default values set on that element's parent element and some don't inherit those properties.

Specificity in CSS

Specificity decides how the browser decides which rules apply when properties have multiple selectors to them. This can be seen when using the class attribute or the ID attribute. They both have multiple selectors but it works differently.

ID and Class Attributes

ID and Class attributes works as selectors and we can define the parameters. An ID attribute only identifies one element whereas a class attribute identifies more attributes. A class selector is a name preceder with a full stop and an ID selector is a name preceded with a hashtag.
